# Adult Income Prediction Project 🚀

Is project mein maine **Machine Learning** ka use karke ye predict kiya hai ki kisi insaan ki saalana income **$50K se zyada** hai ya nahi, unki demographic information (age, education, occupation, etc.) ke aadhar par.

## 📊 Project Overview
- **Model:** Random Forest Classifier (ya jo bhi aapne final use kiya ho).
- **Accuracy:** ~84.8% (Aapne notebook mein ye achieve kiya hai).
- **Deployment:** FastAPI ka use karke ek API banayi gayi hai (`App.py`).

## 📁 File Structure
- `Adult_Prediction.ipynb`: Model training aur Data Analysis ki Jupyter Notebook.
- `App.py`: FastAPI backend code prediction ke liye.
- `adult_model.pkl`: Trained ML model file.
- `Adult dataset/`: Raw data files.
- `requirements.txt`: Project ki saari libraries ki list.
